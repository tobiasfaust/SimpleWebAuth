# users-Verzeichnis

In diesem Ordner speichert SimpleWebAuth lokale Benutzerdaten für die Authentifizierung (z. B. Account-Metadaten, Schlüssel oder Tokens). Die Anwendung liest diese Dateien zur Anmeldung und Verwaltung von Benutzerkonten.

Hinweise:
- Zugriff streng einschränken (Dateirechte: nur Server/Administrator).
- Nicht öffentlich ausliefern und nicht ins Versionskontrollsystem committen.
- Pflege der Einträge erfolgt über die Admin-Tools/Skripte der Anwendung.
- Format und Struktur der Dateien sind in der Projektkonfiguration dokumentiert.
- Regelmäßig sichern und Backups vertraulich behandeln.
